
# GBA-Graphics-and-Collision-Game
A Game Boy Advance (GBA) application developed in C using DMA graphics rendering, object collision detection, and real-time button input handling. The project demonstrates low-level graphics programming concepts including screen synchronization, image rendering, movement systems, and gameplay state management.

# Please see the link to see the demo of the game

https://youtu.be/5IM-fgH2LuA

# **Features**
- Used 3 different images rendered with DMA
- Added two fullscreen background screens
- Added player movement in both x and y directions
- Implemented collision detection between objects
- Added score/timer text updates during gameplay
- Added reset functionality using the SELECT button
- Used keyboard/button input to control gameplay and screen changes

- Clone the repository
